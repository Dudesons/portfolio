  • Netpolling




    5,5 mois

    Create a network scanner that will enable the mapping of the network with a dynamic web interface.
    With this tool, the user can :
    • See the bandwith conssumed
    • Listen the network trafic of a machine
    • Scan the machine in order to know the Operating System and their network ports of the target
    • Move in the network map, like in google map (left, right, top, bottom, zoom, unzoom)
    • Save the actual dynamic map
    background story : I'm a network administrator who fails to quickly see which uses the bandwidth of my network

  • Secure My Linux




    5,5 mois

    This solution is designed for people with no or little knowledge of computer security and have a Linux server.
    The product enables secure a Linux server in a simple, intuitive and without being intrusive via a web interface guiding the user on several points:
    • Analysis of the different services installed to detect security vulnerabilities (eg: apache, mysql, ssh ...)
    • Management systems packages updates
    • Testing the robustness of a password by brute force
    • Information on the firewall, its rules and network traffic
    • Information about the actions of such tools like fail2ban
    Cette solution est destinée à des personnes n'ayant pas ou peu de connaissances en sécurité informatique et qui ont un serveur Linux.
    Le produit permet de sécuriser un serveur linux de façon simple, intuitive et sans être intrusive par le biais d'une interface web guidant l'utilisateurs sur plusieurs points:
    • Analyse des différents services installés afin de détecter des failles de sécurité (ex: apache, mysql, ssh ...)
    • Gestion des mises à jour des paquets systèmes
    • Test de la robustesse des mot de passe par brute force
    • Informations sur le pare-feu, ses régles et le traffic réseau
    • Informations sur les actions d'outil tel que fail2ban


    Technologies :

    Perl JavaScript IpTables

    John the Ripper Fail2Ban Linux

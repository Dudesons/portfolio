Netpolling




Create a network scanner that will enable the mapping of the network with a dynamic web interface.
With this tool, the user can :
  • See the bandwith conssumed
  • Listen the network trafic of a machine
  • Scan the machine in order to know the Operating System and their network ports of the target
  • Move in the network map, like in google map (left, right, top, bottom, zoom, unzoom)
  • Save the actual dynamic map
background story : I'm a network administrator who fails to quickly see which uses the bandwidth of my network

  • Implementation of web-worker technologies and Ajax
  • Setting up the database with an ORM (object-relational mapping)
  • Establishment of parallelism of scanning script
